# 2조 T2
이번 주제는 "종강" 이었고 팀원들끼리 종강을 염원하는 마음으로 종강까지 버티는 타임어택 미니게임을 개발했습니다.

게임의 형식은 고전 똥피하기 게임을 참고했고, 로컬 컴퓨터에 저장되어 있는 교수님의 사진을 사용하는 기능을 통해 유저 커스터마이징이 가능한 서비스를 개발하고자 했습니다.
